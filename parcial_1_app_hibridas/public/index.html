API de Gestión Névola

Documentación completa de la API para gestionar redes de líderes y colaboradores de Venta Directa.

Endpoints de Usuarios

GET /api/users Obtener todos los usuarios

Respuestas

Código Descripción
200 Lista de usuarios obtenida exitosamente

Ejemplo de respuesta

    {
        "users": [
            {
                "id": "680b952779bf2d7ee7a386be",
                "name": "john",
                "email": "john@mail.com"
            },
            {
                "id": "6810d327b7297b29bec331a6",
                "name": "doe",
                "email": "doe@mail.com"
            }
        ]
    }
POST /api/users Crear un nuevo usuario

Request Body

Campo Tipo Requerido Descripción
name string Sí Nombre completo del usuario
email string Sí Correo electrónico del usuario
password string Sí Contraseña del usuario

Ejemplo de request

    {
        "name": "Juan Perez",
        "email": "juan.perez@example.com",
        "password": "123456"
    }

Respuestas

Código Descripción
201 Usuario creado exitosamente
400 Datos de entrada inválidos
403 Faltan parámetros

Ejemplo de respuesta

    {
        "msg": "Usuario creado",
        "data": {
            "id": "6818378a2012a41414ff4589",
            "name": "Juan Perez"
        }
    }

Parámetros de ruta

    }

Endpoints de Productos

GET /api/products Obtener todos los productos

Parámetros de consulta (opcionales)

Parámetro Tipo Requerido Descripción
name string No Texto para buscar en el nombre de los productos, puede ser parcial
category number No Filtra productos por categoría
minPrice number No Precio mínimo de los productos
maxPrice number No Precio máximo de los productos

Respuestas

Código Descripción
200 Lista de productos obtenida exitosamente
500 Error del servidor

Ejemplo de respuesta

{
    "products": [
        {
            "id": "681855e9c8af13c86ecc332b",
            "name": "Producto 1",
            "description": "Descripción del producto 1",
            "category": 1,
            "price": 100
        },
        {
            "id": "681855e9c8af13c86ecc332c",
            "name": "Producto 2",
            "description": "Descripción del producto 2",
            "category": 2,
            "price": 200
        }
    ]
}

POST /api/products Crear un nuevo producto

Request Body

Campo Tipo Requerido Descripción
name string Sí Nombre del producto (3-50 caracteres)
description string Sí Descripción del producto (10-200 caracteres)
category number Sí ID de la categoría del producto
price number Sí Precio del producto (mayor a 0)

Ejemplo de request

    {
        "name": "Producto nuevo",
        "description": "Descripción detallada del producto nuevo",
        "category": 1,
        "price": 150.50
    }

Respuestas

Código Descripción
201 Producto creado exitosamente
400 Datos de entrada inválidos
403 Faltan parámetros

Ejemplo de respuesta

    {
        "msg": "Producto creado",
        "data": {
            "id": "681855e9c8af13c86ecc332b",
            "name": "Producto nuevo",
            "category": 1,
            "price": 150.50
        }
    }
